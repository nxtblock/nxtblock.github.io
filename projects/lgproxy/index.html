GenGen 洛谷国际

基于 CloudFlare 的洛谷国际版加速服务，为国内用户提供快速、无广告的洛谷国际版访问体验

网站代理 部署于 CloudFlare 无广告，速度极快 成立于 2024-07-28

项目简介

GenGen 洛谷国际是专为国内编程竞赛爱好者设计的洛谷国际版（luogu.com）加速服务。通过全球分布的 CloudFlare 节点，我们提供了一个访问速度更快、无广告干扰的洛谷国际版访问渠道，让国内用户也能享受流畅的编程竞赛学习体验。

项目历史

GenGen 洛谷国际项目成立于2024年7月28日，至今经历了两个重要的发展阶段：

  • 第一阶段（2024-07-28 至 2025-08-10）：由 general0826 领导的"用创意改变生活"时代。在此期间，项目专注于基础架构的搭建和核心代理功能的实现，奠定了稳定的技术基础。
  • 第二阶段（2025-08-11 至今）：由 Gcend 领导的"用科技改变生活"时代。在此期间，项目进行了重大技术重构和功能优化，显著提升了访问速度和用户体验。

技术实现

我们利用 CloudFlare Workers 的强大功能，实现了对洛谷国际版的高效代理。通过全球分布的边缘节点，用户请求可以就近处理，大大减少了网络延迟。同时，我们移除了原站的广告内容，为用户提供纯净的学习环境。

功能特点

  • 访问速度：极快（Cloudflare 全球边缘网络，平均响应时间低于100ms）
  • 广告：完全无广告，提供纯净的学习环境
  • 功能范围：完整代理 luogu.com（用户/专栏等），支持所有功能
  • 使用说明：只需将原洛谷国际站链接中的域名替换为 lg.gengen.qzz.io 即可

使用方法

使用 GenGen 洛谷国际非常简单，您只需要将原洛谷国际站链接中的域名替换为 lg.gengen.qzz.io 即可。例如，如果原链接是 https://www.luogu.com/problem/P1001，您只需访问 https://lg.gengen.qzz.io/problem/P1001 即可获得加速访问体验。
