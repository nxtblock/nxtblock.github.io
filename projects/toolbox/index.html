GenGen ToolBox

C++ raylib 桌面应用

项目简介

GenGen ToolBox 是一款基于 raylib、使用 C++ 开发的桌面工具箱，专为 OI（信息学奥林匹克竞赛）选手设计。它集成了导航、签到、MC 启动、脚本市场等多种实用功能，通过模块化设计提供一站式服务，便于用户根据需求自由扩展。

核心特性

  • 模块化架构：用户可以根据自己的需求启用或禁用特定功能模块，避免功能冗余，保持界面简洁。
  • 一站式服务：整合了从日常签到到工具启动的多种功能，减少在不同应用间切换的时间成本。
  • 跨平台潜力：基于 raylib 图形库开发，界面轻量且具备良好的跨平台扩展能力。
  • 持续更新：团队定期发布更新，添加新功能并优化用户体验。
  • 用户友好：界面直观，操作简单，适合各个水平的 OI 选手使用。

功能模块

导航模块

提供常用 OI 站点的快速访问入口，包括洛谷、Codeforces、AtCoder 等，方便用户快速跳转到相关页面。

签到模块

自动完成洛谷、力扣等平台的每日签到任务，帮助用户养成持续学习的习惯。

MC 启动器

集成 Minecraft 启动器功能，方便用户在学习之余放松娱乐。

脚本市场

提供各类编程竞赛相关的脚本下载，用户可以根据需要安装和使用。

使用指南

  1. 从下方下载链接获取最新版本的 GenGen ToolBox 安装包
  2. 运行安装程序，按照提示完成安装过程
  3. 启动应用，根据引导完成初始配置
  4. 在设置中启用需要的功能模块
  5. 开始享受一站式 OI 工具体验
