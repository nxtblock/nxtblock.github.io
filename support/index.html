支持我们

我们在课业之余抽空开发与维护开源项目。你的支持能帮助我们覆盖基础成本、持续打磨体验。 作为一支由学生组成的团队，我们致力于为编程竞赛社区提供高质量的开源工具。 每一份支持都是对我们工作的认可和鼓励，让我们能够更好地专注于产品开发和用户体验优化。

赞赏通道

扫码赞赏，感谢你的每一份支持。你的赞赏将用于服务器成本、开发工具订阅等项目维护开支，帮助我们持续提供稳定的服务。

打开大图

你还能这样帮助

  • Star/Watch：关注 GitHub 仓库，第一时间获得更新。给个 Star 吧🙏，这是对我们开源项目最大的鼓励。
  • 反馈建议：提交 Issue 共享想法与使用场景，帮助我们改进产品功能。
  • 传播分享：推荐给身边的 OIer 与同学，扩大我们的用户群体。
  • 贡献代码：欢迎 PR 与脚本/模块扩展，共同完善我们的项目。
  • 撰写文档：帮助完善项目文档和使用教程，让新用户更容易上手。
  • 测试反馈：参与新功能测试，提供详细的使用反馈和建议。

你的支持意味着什么

作为一支由学生组成的开源团队，我们利用课业之余的时间开发和维护项目。你的支持不仅能够帮助我们覆盖服务器、域名等基础成本，更重要的是给予我们继续前行的动力。

每一份 Star、每一次分享、每一条反馈建议，都是我们持续改进产品的动力源泉。我们相信，通过社区的共同支持，能够为编程竞赛爱好者打造更优秀的工具。

你的支持将主要用于：

  • 服务器和域名费用：确保项目稳定运行
  • 开发工具订阅：提升开发效率和代码质量
  • 学习资源：持续提升技术能力，开发更优秀功能
  • 社区活动：组织线上交流，收集用户反馈

合作伙伴

我们感谢以下平台和组织对我们的支持：

  • GitHub：为我们提供代码托管和协作开发平台
  • CloudFlare：为我们的项目提供 CDN 和安全服务
  • 洛谷：作为重要的编程竞赛学习平台，与我们共同服务 OI 社区

如果你或你的组织希望与我们合作，请通过 GitHub 或洛谷队联系我们。

特别感谢

我们要特别感谢所有支持和使用我们项目的用户，以及提供反馈和建议的社区成员。你们的鼓励是我们持续前进的动力。

同时感谢所有为项目贡献代码、文档或想法的开源社区成员，以及在洛谷、GitHub 等平台与我们交流的朋友们。
